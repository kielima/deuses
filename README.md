# Deuses — Dashboard Mundial de Mitologia

Dashboard interativo com **3.738 deidades** de **312 culturas** ao redor do mundo.

## Funcionalidades

- Busca por nome de deidade
- Filtros por cultura/panteão, gênero e geografia
- Gráfico de top 25 culturas por número de deidades
- Distribuição por gênero (masculino/feminino/andrógino)
- Top 15 geografias
- Tabela paginada e ordenável com 50 registros/página

## Dados

Base de dados *Gods and Goddesses* cobrindo 312 panteões de todos os continentes: Grego/Romano (360), Hindu (237), Shinto (219), Celta (179), Romano (153), Egípcio (146), Maia (142), Asteca (124), Lituano (111), Chinês (92), e muitos outros.

## Como usar

Abra `index.html` diretamente no navegador — funciona offline, todos os dados estão embutidos no arquivo.

## Tecnologia

HTML auto-contido · [Chart.js](https://www.chartjs.org/) para visualizações · Vanilla JS
